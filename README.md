# Team Invitations

A private, browser-local web app for creating and tracking NGA and USAG team invitations.

## Use

Open the published GitHub Pages app, choose NGA or USAG, and create an invitation. Program Setup stores reusable schedule and cost details for each program. The app can print an invitation, copy its text, and export or import a JSON backup.

## Privacy

Athlete records are saved only in the current browser under `team-invites-v1`. The app has no account, server, analytics, or network-based data storage. No athlete information is included in this repository.

Because browser storage is specific to a browser and device, export a backup before clearing browser data or moving to another device.

## Typography

The app bundles STIX Two Text italic font files from Fontsource. STIX is licensed under the SIL Open Font License 1.1; the license is included at `assets/fonts/OFL.txt`.
